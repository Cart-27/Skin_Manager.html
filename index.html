⚙ One-Time Setup — Cloudflare Worker Proxy

Steam blocks direct browser requests (CORS). A free Cloudflare Worker acts as a middleman — it's a 5-minute setup, free forever (100k requests/day), no credit card needed.

1
Create a free Cloudflare account
dash.cloudflare.com/sign-up — no credit card required
2
Create a Worker

In the Cloudflare dashboard: Workers & Pages → Create → Create Worker. Name it anything (e.g. steam-proxy). Click Deploy.

Then click Edit Code and replace everything with this:

export default {
  async fetch(request) {
    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: corsHeaders() });
    }
    const body = await request.text();
    const steam = await fetch(
      'https://api.steampowered.com/ISteamRemoteStorage/GetPublishedFileDetails/v1/',
      { method: 'POST', body }
    );
    const data = await steam.json();
    return new Response(JSON.stringify(data), {
      headers: { 'Content-Type': 'application/json', ...corsHeaders() }
    });
  }
};
function corsHeaders() {
  return {
    'Access-Control-Allow-Origin': '*',
    'Access-Control-Allow-Methods': 'POST, OPTIONS',
    'Access-Control-Allow-Headers': 'Content-Type',
  };
}

Click Save and Deploy.
