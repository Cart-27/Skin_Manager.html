⚙ Cloudflare Worker Setup

Steam blocks direct browser requests. A free Cloudflare Worker proxies the calls. 5-minute setup, 100k requests/day free.

1
Create a free Cloudflare account
dash.cloudflare.com/sign-up
2
Deploy the worker via terminal

Install Node.js from nodejs.org, then run these commands:

npm install -g wrangler
wrangler login
wrangler deploy steam-proxy-worker.js --name steam-proxy --compatibility-date 2024-01-01

Download the worker file: steam-proxy-worker.js

3
Paste your Worker URL
